# Demo Scope

## Purpose

This document defines the browser-playable seven-day vertical slice demo for portfolio/application review.

This is not the full game and not the final production MVP.

## Target

The demo should make players spend around 15–20 minutes in Café Apokalypso and want to know what happens on Day 8.

## Scope

The demo covers the first 7 in-game days.

Players should be able to:
- serve guests
- manage simple resources
- adjust prices or offers
- buy basic ingredients
- clean the café
- run small ads
- experience scripted guest and event moments
- notice escalating weirdness
- reach a clear Day 7 cliffhanger

## Weirdness Curve

Days 1–2: mostly normal café management.

Days 3–5: subtle strange hints appear.

Days 6–7: the weirdness becomes clearly noticeable and cannot be missed.

## Day 7 Cliffhanger

The demo should end with a clear hook:

Something is wrong with this café.

The intended player reaction is:

“I want to know what happens on Day 8.”

## Out of Scope

- backend
- accounts
- real AI API
- analytics/tracking
- downloadable desktop app
- full staff system
- complex economy simulation
- multiple locations
- full apocalypse-management layer
- final production art
